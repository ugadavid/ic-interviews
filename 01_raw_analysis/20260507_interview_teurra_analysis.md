## 3. Pratiques pédagogiques d’intercompréhension

### Vision de l’intercompréhension

L’intercompréhension est envisagée comme un **outil de transfert de connaissances linguistiques**.

Exemples :

* comparaisons lexicales (suffixes diminutifs portugais / espagnol / français) ;
* rapprochements grammaticaux (ser/estar) ;
* travail plurilingue sur textes et audio ;
* recours aux langues déjà connues comme support d’apprentissage.

Approche très **souple, contextualisée et intuitive**.

L’IC n’est pas traitée comme un objet isolé, mais intégrée à une pédagogie des langues plus globale.

### Place du plurilinguisme

Le plurilinguisme est présenté comme :

* naturel ;
* désinhibant ;
* réaliste socialement.

Les étudiants sont encouragés à mobiliser leurs répertoires linguistiques existants.

L’enseignante cherche explicitement à déconstruire :

* le mythe du natif ;
* la rigidité monolingue ;
* la hiérarchie implicite entre variétés linguistiques.

## 4. Usages numériques concrets

### Dispositif principal développé

Création d’un site pédagogique (« Portugais entre langues ») combinant :

* podcasts ;
* dialogues audio ;
* activités interactives autocorrectives ;
* fiches pédagogiques ;
* travail autonome à domicile.

Organisation par épisode :

1. écouter ;
2. comprendre ;
3. pratiquer ;
4. approfondir ;
5. interagir.

### Particularités du dispositif

* forte cohérence pédagogique ;
* souci esthétique affirmé ;
* intégration native du plurilinguisme ;
* multimodalité (texte + audio + interaction).

Les étudiants utilisent le dispositif dans des contextes variés (métro, bus, travail autonome).

## 5. Difficultés identifiées

Contrairement à d’autres enseignants, Teurra ne rapporte **pas de difficulté technique majeure**.

Les contraintes évoquées sont surtout :

### 1. Temps et charge de travail

Production extrêmement chronophage.

### 2. Contraintes institutionnelles

* absence initiale de financement ;
* nécessité de légitimation institutionnelle ;
* besoin de publication académique.

### 3. Passage à l’échelle

Difficulté à mutualiser ou industrialiser les contenus produits individuellement.

## 6. Besoins exprimés (signal fort)

### Besoin central émergent :

**Banque mutualisée de contenus transformables pédagogiquement.**

Teurra imagine un système permettant :

* de sélectionner un texte ou audio ;
* de choisir les langues ;
* de choisir l’objectif pédagogique ;
* de générer automatiquement différentes formes d’activités.

Exemples souhaités :

* travail lexical ;
* grammaire ;
* compréhension ;
* lexique transparent ;
* écoute/lecture ;
* comparaison plurilingue.

### Vision exprimée

Un même contenu pourrait être :

> transformé selon l’objectif pédagogique choisi.

Ce besoin dépasse la simple logique « outil auteur ».

Il combine :
**contenu mutualisé + transformation didactique + modularité pédagogique.**

## 8. Signaux importants pour la recherche

### Signal 1 — Tous les enseignants n’ont pas les mêmes besoins

Teurra représente un profil **très autonome**, atypique.

Elle ne demande pas de “faire à sa place”.

Elle cherche plutôt :

> un multiplicateur de puissance pédagogique.

### Signal 2 — Le besoin ne porte pas seulement sur les outils

Le besoin porte aussi sur :

> des ressources mutualisées prêtes à transformer.

### Signal 3 — Validation indirecte d’une architecture type BHE

Son discours rejoint fortement l’idée de séparation :

**contenu pédagogique**
↔
**objectif didactique**
↔
**forme d’interaction**

### Signal 4 — Forte importance du multimodal

Audio + texte + interaction + plurilinguisme apparaissent centraux.

## 10. Citation synthétique de l’entretien

L’entretien de Teurra suggère qu’un outil pertinent en intercompréhension ne devrait pas seulement permettre de créer des activités, mais aussi offrir une base mutualisée de contenus transformables selon différents objectifs pédagogiques, dans un environnement simple, multimodal et plurilingue.
