## 4. Place du plurilinguisme

L’approche plurilingue est :

> active et interactionnelle.

L’objectif n’est pas uniquement :

> comprendre d’autres langues romanes.

Mais :

> apprendre à produire une langue compréhensible.

Concept central :

### interproduction

Définition implicite :

> simplifier sa langue afin qu’elle soit accessible aux autres langues romanes.

Compétences travaillées :

* ralentissement du débit ;
* vocabulaire transparent ;
* évitement des mots opaques ;
* reformulation ;
* explicitation ;
* stratégies compensatoires.

---

## 6. Difficultés identifiées

### 1. Difficultés techniques étudiantes

Signal fort du corpus.

Constat :

les étudiants rencontrent des difficultés importantes avec :

* enregistrement vidéo ;
* qualité audio ;
* affichage grille ;
* export ;
* manipulation d’outils numériques.

Observation critique :

> les « digital natives » ne maîtrisent pas nécessairement les outils.

Conséquence :

fort besoin :

> d’environnements simplifiés et intégrés.

Idéal décrit :

> un outil “one-click”.

---

### 2. Manque d’outils adaptés à l’oral plurilingue

Très gros signal.

Absence actuelle :

> d’outils intégrés pour interaction orale IC.

Besoins exprimés :

* enregistrement automatique ;
* grille vidéo par défaut ;
* transcription multilingue ;
* tagging ;
* intégration Moodle.

---

### 3. Travail colossal de transcription

Signal majeur.

Méthode actuelle :

### Speechmatics

pour transcription multilingue.

Puis :

### ELAN

pour annotation manuelle.

Étiquetages :

* langue ;
* parties du discours ;
* actes de parole ;
* stratégies IC ;
* reformulations ;
* pauses ;
* focalisations.

Charge de travail :

> environ 20 heures pour 20 minutes d’interaction.

Corpus actuel :

> environ 150 heures enregistrées.

Très fort frein scientifique et pédagogique.

---

### 4. Difficulté de l’interproduction

Constat très important :

> les étudiants comprennent mieux qu’ils ne savent se faire comprendre.

L’interproduction est identifiée comme :

> difficulté centrale.

Problème :

les étudiants produisent souvent :

* mots opaques ;
* formulations non transparentes ;
* débit trop rapide.

## 7. Rapport à l’intelligence artificielle

Positionnement :

**très ouvert, pragmatique et orienté recherche.**

Idée particulièrement forte :

> entraîner une IA sur un corpus réel d’interactions IC.

Objectifs imaginés :

### compréhension orale

entraîner à reconnaître :

* stratégies ;
* transparence ;
* compréhension.

### interproduction

entraîner à :

* simplifier sa langue ;
* choisir mots transparents ;
* produire du compréhensible.

Très forte convergence avec :

> apprentissage assisté de l’intercompréhension.

Critique intéressante :

les IA généralistes actuelles :

> ne maîtrisent pas spontanément les principes IC.

Exemple :

production de textes insuffisamment transparents pour toutes les langues romanes.

Hypothèse forte :

> un corpus IC + les sept tamis pourraient améliorer fortement les résultats.
