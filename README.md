# Intercompréhension – Analyse des entretiens enseignants (Projet ALPAGA / REPLI4C)

## Contexte

Ce dossier regroupe une première vague d’analyses qualitatives d’entretiens menés auprès d’enseignant·es impliqué·es, à différents degrés, dans des pratiques d’intercompréhension (IC) entre langues romanes.

Ces entretiens ont été réalisés dans le cadre du projet **ALPAGA / REPLI4C**, avec un double objectif :

1. **Comprendre les pratiques réelles des enseignant·es**
2. **Identifier des besoins concrets susceptibles d’orienter la conception de futurs outils ou prototypes pédagogiques**

L’approche adoptée est volontairement :

> exploratoire, qualitative et orientée design pédagogique.

L’objectif n’est pas de produire une vérité générale sur l’intercompréhension, mais :

> de comprendre ce que le terrain raconte.

## Positionnement méthodologique

Les entretiens sont de type :

> **semi-directifs à visée exploratoire**

Cependant, plusieurs entretiens ont progressivement évolué vers des moments de :

> **co-idéation pédagogique**

Certain·es participant·es ont spontanément réagi à des idées de prototypes, proposé des extensions ou réfléchi collectivement à des solutions possibles.

Cela ne constitue pas un biais à éliminer, mais :

> un matériau pertinent dans une démarche orientée conception pédagogique.

Le corpus doit donc être lu à travers une double entrée :

### 1. Compréhension du terrain

Que font réellement les enseignant·es ?

### 2. Projection design

Quels besoins, tensions ou opportunités émergent pour de futurs outils ?

---

## Principe d’analyse

Important :

Les analyses doivent privilégier :

### les convergences

sans écraser :

### les contradictions

L’objectif n’est pas de produire artificiellement un consensus.

Au contraire :

les tensions, divergences et désaccords sont considérés comme particulièrement féconds.

Exemples de tensions déjà émergentes :

* compréhension ↔ interproduction
* IC réceptive ↔ IC interactionnelle
* ludique ↔ gamification
* autonomie ↔ scénarisation guidée
* spécialiste IC ↔ enseignant non spécialiste
* activités longues ↔ micro-activités intégrables
* oral ↔ textuel
* théorie ↔ pratiques écologiques spontanées

## État actuel : hypothèses émergentes (non stabilisées)

Les éléments suivants semblent apparaître de manière récurrente dans le corpus, sans être considérés comme définitivement établis :

### Signaux forts observés

* importance croissante de l’oral ;
* manque d’outils adaptés à l’interproduction ;
* besoin de mutualisation des ressources ;
* intérêt fort pour des outils simples, modulaires et rapides à prendre en main ;
* besoin de multimodalité ;
* importance du gain de temps enseignant ;
* forte valeur des supports authentiques ;
* intérêt pour les activités contextualisées et finalisées.

### Signaux émergents

* intercompréhension comme désinhibition linguistique ;
* sentiment d’auto-efficacité (“super-pouvoirs linguistiques”) ;
* importance des biographies langagières ;
* réflexion sur des niveaux de progression en IC ;
* micro-grammaires émergentes ;
* rôle des images dans la réparation communicationnelle ;
* IA spécialisée en intercompréhension.

Ces hypothèses restent :

> provisoires et ouvertes à révision.

## Philosophie générale

Ce corpus ne cherche pas uniquement à décrire des pratiques existantes.

Il vise également à explorer :

> ce que pourrait devenir l’intercompréhension outillée.

Avec une attention particulière portée à :

* l’oral ;
* l’interproduction ;
* la multimodalité ;
* les ressources mutualisées ;
* les contextes réels d’usage ;
* les besoins concrets des enseignant·es.

Le principe directeur reste :

> écouter attentivement le terrain avant de construire.
