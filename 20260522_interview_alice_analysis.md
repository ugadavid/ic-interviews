> compréhension des langues romanes.

Travail sur :

* compréhension écrite ;
* compréhension orale ;

* langues romanes de l’Alliance UNITA.

### Semestre 2

Focus :

> interaction et interproduction.

Travail sur :

* stratégies interactionnelles ;
* verbal ;
* paraverbal ;
* non verbal ;
* ralentissement du débit ;
* segmentation ;
* reformulation ;
* clarification.

L’objectif est :

> rendre les étudiants capables d’interagir réellement en intercompréhension.

## 4. Place du plurilinguisme

Le plurilinguisme est présenté comme :

> pratique réelle de communication.

Particularité forte :

les étudiants restent dans leur langue.

Compétence explicitement nommée :

> interproduction.

Objectif :

apprendre à :

* choisir des mots transparents ;
* simplifier sans appauvrir ;
* adapter sa langue ;
* rendre son discours accessible.

Point clé :

> montrer qu’il est possible de communiquer efficacement sans langue commune unique.

## 6. Difficultés identifiées

### 1. Manque d’outils IC spécialisés

Signal extrêmement fort.

Constat :

> il existe des manuels IC

mais peu :

> d’activités numériques réellement adaptées.

Besoins identifiés :

* tableaux plurilingues ;
* tamis interactifs ;
* correspondances panromanes ;
* activités de comparaison ;
* oral ;
* spécialisation professionnelle.

---

### 2. Difficulté de mutualisation

Production actuellement :

> très artisanale.

Les enseignants créent eux-mêmes :

* leurs exercices ;
* leurs corpus ;
* leurs scénarios.

Forte demande implicite :

> mutualisation interuniversitaire.

---

### 3. Manque de matériel oral didactisé

Signal majeur.

Besoin :

> oral authentique mais pédagogiquement exploitable.

Problèmes actuels :

* qualité audio ;
* absence de sous-titres ;
* manque de transparence ;
* débit trop rapide ;
* erreurs étudiantes.

---

### 4. Absence de ressources pour langues de spécialité

Très gros signal.

Besoins spécifiques :

* entreprise ;
* action humanitaire ;
* accueil international ;
* médiation ;
* communication professionnelle.

Constat :

> peu de ressources IC contextualisées professionnellement.

---

## 7. Besoins exprimés (signaux forts)

### Signal 1 — Vidéos IC didactisées

Besoin majeur :

> corpus vidéo d’interactions réelles en intercompréhension.

Critères :

* experts IC ;
* bonne qualité audio ;
* sous-titres ;
* stratégies visibles ;
* segmentation ;
* adaptation du débit.

Idée forte :

filmer :

> de vraies réunions d’enseignants experts en intercompréhension.

Objectif :

montrer aux étudiants :

> qu’une communication plurilingue réelle est possible.

---

### Signal 2 — Outils multimédias interactifs

Forte demande pour :

* vidéo ;
* oral ;
* sous-titres interactifs ;
* reconnaissance vocale ;
* audio multilingue.

Exemple concret :

lecture correcte de textes dans des langues moins maîtrisées
(catalan, occitan, roumain).

Besoin explicite :

> accès aux sons des langues.

---

### Signal 3 — Activités IC spécialisées

Besoin fort de :

> scénarios contextualisés.

Exemples :

* vendre un produit ;
* accueil d’étudiants internationaux ;
* communication professionnelle ;
* réunions plurilingues ;
* mails simplifiés.

Orientation :

**IC appliquée au monde professionnel.**

---

### Signal 4 — Mutualisation intelligente des ressources

Très forte validation d’un :

> corpus partagé intercompréhension.

Réaction particulièrement positive au prototype des tamis.

Vision projetée :

* textes mutualisés ;
* annotations communes ;
* réutilisation intertextuelle ;
* enrichissement collectif.

---

### Signal 5 — Travail autonome guidé

Besoin important :

> réviser l’IC hors classe.

L’outil devrait permettre :

* entraînement autonome ;
* réactivation ;
* mise en pratique opérationnelle.

---

### Signal 6 — Immersion culturelle contextualisée

Idée originale émergente :

> explorer des villes à distance.

Objectif :

travailler sur :

* affichage urbain ;
* signalétique ;
* environnement culturel ;
* indices linguistiques authentiques.

Exemple évoqué :
Timisoara via exploration numérique.

Dimension :

> immersion culturelle augmentée.

## 11. Citation synthétique de l’entretien

L’entretien d’Alice suggère qu’un outil pertinent pour l’intercompréhension devrait dépasser les exercices textuels classiques afin de proposer des interactions multimodales, professionnalisantes et scénarisées, soutenues par un corpus mutualisé de ressources authentiques et des outils adaptés à l’oral.
