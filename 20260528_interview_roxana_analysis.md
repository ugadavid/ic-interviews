## 4. Place de la multimodalité

Signal majeur du corpus.

Observation centrale :

les étudiants utilisent spontanément :

> des images pour réparer la communication.

Exemple emblématique :

### l’épisode du “tejocote”

Situation :

des étudiants mexicains tentent d’expliquer une boisson traditionnelle à une étudiante italienne.

Blocage :

fruit culturellement inconnu.

Tentatives initiales :

* descriptions verbales ;
* comparaisons ;
* reformulations.

Résultat :

échec partiel.

Résolution :

> partage immédiat d’une image.

Effet :

> compréhension instantanée.

Constat fort :

les images semblent :

> soutenir naturellement l’intercompréhension.

Réflexion importante de Roxana :

> pourquoi interdire un moyen de communication qui fonctionne ?

Dimension particulièrement forte :

**multimodalité écologique.**

---

## 5. Pratiques pédagogiques concrètes

### Télé-tandem bilingue

Organisation :

échanges :

* français ↔ espagnol.

Règle initiale :

### 30 minutes français

### 30 minutes espagnol

Observation particulièrement intéressante :

progressivement :

> les étudiants cessent spontanément de changer de langue.

Phénomène observé :

certains parlent :

* espagnol

pendant que d’autres répondent :

* français

Sans rupture de communication.

Signal très fort :

> émergence spontanée d’intercompréhension interactionnelle.

Question soulevée :

faut-il :

> imposer le changement de langue ?

Ou :

> laisser émerger les stratégies naturelles ?

---

### Patience communicationnelle

Observation particulièrement riche.

Retour d’étudiants :

> « au début on ne comprend rien, puis ça vient ».

Roxana identifie une compétence implicite :

> apprendre à ne pas paniquer face à l’incompréhension.

Dimension émergente :

**tolérance à l’incertitude linguistique.**

Très forte proximité avec :

> désinhibition linguistique observée chez Thomas.

## 6. Réflexion originale sur les niveaux

Signal particulièrement fort du corpus.

Question centrale posée :

> peut-il exister des niveaux d’intercompréhension ?

Parallèle proposé :

### CECRL débutant (A1–A2)

Compétences :

* situations concrètes ;
* vocabulaire fréquent ;
* objets tangibles ;
* communication immédiate.

Exemples :

* nourriture ;
* transport ;
* santé ;
* besoins quotidiens.

Constat :

> l’intercompréhension semble très bien fonctionner.

---

### CECRL avancé (B2–C1)

Compétences :

* abstraction ;
* argumentation ;
* concepts complexes ;
* discours académique.

Question forte :

> l’intercompréhension fonctionne-t-elle encore à ces niveaux ?

Exemple personnel :

lecture d’ouvrages complexes d’Umberto Eco en italien :

> compréhension relativement réussie.

Mais :

> difficulté anticipée dans une interaction abstraite orale complexe.

Question émergente majeure :

> les limites cognitives de l’intercompréhension varient-elles selon les niveaux discursifs ?

Signal potentiellement très important pour la recherche.
